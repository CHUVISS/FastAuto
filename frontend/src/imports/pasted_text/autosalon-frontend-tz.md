Техническое задание для фронтенда автосалона
1. Концепция
Современный минималистичный дизайн как у Auto.ru — чистый, функциональный, без лишних элементов. Фокус на удобстве пользователя и скорости работы.
2. Основные страницы
2.1. Главная страница
•	Герой-баннер с поиском авто
•	Категории автомобилей (новые/с пробегом)
•	Популярные модели
•	Преимущества салона
•	Форма быстрой заявки
2.2. Каталог автомобилей
Фильтры (как на Auto.ru):
•	Цена (от/до)
•	Марка/модель
•	Год выпуска
•	Пробег
•	Тип КПП (автомат/механика)
•	Тип топлива
•	Цвет
•	Объем двигателя
•	Привод
•	Кузов
Отображение:
•	Сетка карточек (3-4 в ряд на десктопе)
•	Быстрый просмотр при наведении
•	Сортировка: по цене, году, дате добавления
2.3. Карточка автомобиля
Структура:
1.	Галерея фото (до 10 изображений, слайдер)
2.	Основная информация:
•	Цена
•	Марка, модель, год
•	Пробег
•	Двигатель, КПП, привод
•	Цвет
3.	Характеристики (таблица)
4.	Описание
5.	Контакты менеджера
6.	Форма записи на просмотр
2.4. Личный кабинет
Для клиентов:
•	Мои заявки
•	История просмотров
•	Сохраненные авто
•	Настройки профиля
Для менеджеров:
•	Мои клиенты
•	Сделки в работе
•	Календарь просмотров
•	Статистика
2.5. Панель управления (админка)
•	Управление каталогом
•	Добавление/редактирование авто
•	Заявки от клиентов
•	Управление пользователями
•	Отчеты
3. Технические требования
3.1. Стек технологий
React 18+ / Next.js 14
TypeScript
Tailwind CSS
React Query / SWR
React Hook Form + Zod
Framer Motion (анимации)
3.2. Производительность
•	Первая загрузка < 2.5 сек
•	LCP < 2.5 сек
•	CLS < 0.1
•	Оптимизация изображений (WebP, lazy loading)
•	Кэширование API запросов
3.3. Адаптивность
•	Mobile First подход
•	Брейкпоинты: 320px, 768px, 1024px, 1280px
•	Touch-friendly интерфейс

12. Что должен понять фронтендер
1.	Дизайн: Современный и минималистичный как Auto.ru
2.	Фокус: Удобство пользователя и скорость
3.	Стек: React + TypeScript + Tailwind
4.	Главное: Каталог с фильтрами и карточки авто
5.	Цель: Сделать так, чтобы клиент легко нашел и купил авто
 
Критерий успеха: Пользователь заходит на сайт → находит авто за 3 клика → оставляет заявку. Все быстро, понятно, без лишних действий.
